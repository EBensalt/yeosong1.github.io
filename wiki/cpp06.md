이 리터럴은 다음 스칼라 유형 중 하나에 속해야 합니다: char, int, float 또는 double.

char literal values 예시 'c', 'a'...
(논프린터블은 인자로 안받는 걸로.)

int literal values 예시 0, -42, 42...

float literal values: 0.0f, -4.2f, 4.2f... -inf, +inf, nan




double literal values: 0.0, -4.2, 4.2...
